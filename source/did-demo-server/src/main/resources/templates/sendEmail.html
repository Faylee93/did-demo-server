Issue a mobile ID

To issue a mobile ID, please complete the issuance using the following methods.

  • Enter the DID of OpenDID and send a push to the app.
  • Select the notification on your phone, run the OpenDID app, and proceed with the issuance.
My email address

The QR code will be sent to your email.

Please complete the issuance of the certificate by scanning the QR code after running the app.

If the sent email is not received, please check the spam mailbox.

The QR code is valid for 7 days after sending.
